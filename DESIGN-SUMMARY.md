# Design summary

- Design concept: Accounts payable control
- Layout personality: invoice operations
- Typography: Roboto Slab headings with Roboto body text
- Color palette: primary #0f172a, secondary #eab308, accent #ef4444, background #fefce8, text #1c1917, muted #71717a
- Shape language: Sharp corners
- Visual motif: invoice strips and approval stamps
- Unique layout choices: first sections use AP control room, then a custom service index, process ribbon, question block, and resource strip with varied widths and alignments.
- Difference from previous sites: this redesign changes the hero composition, section ordering, image treatment, card style, spacing rhythm, button shape, and footer treatment for this domain.
